SQL Update Job Creator

This tool generates a SQL update job script to be used on c9980 web pages.

Important guidelines for usage:

PLEASE CONSIDER THESE GUIDELINES WHEN USING THIS JOB CREATOR:

  • The SQL file must have each query on a single line - each ending with a semi-colon.
  • The SQL file may not contain any empty lines - even at the end of the file.
  • The job name must be a friendly name - with no spaces and:
    • Not start with 'job.'
    • Not end with '.sh' or any file extensions.
  • Copy your job to c9980:/home/ucsretail/jobs.
